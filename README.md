## Wound Image Classification

### Code Requirements
- Flask(`pip install flask`)
- keras(`pip install keras`)
- Numpy(`pip install numpy`)


### What steps you have to follow??

- Run `app.py`
- Go to ` http://127.0.0.1:5000/Image_Classification `
